Действия по регистрации
    Через SQL создается SUPER_ADMIN
    Регистрируется пользователь с ролью OWNER
    Регистрируется пользователь с ролью CLIENT

Действия в системе
SUPER_ADMIN
    создаёт и удаляет админов

ADMIN
    Имеет возможность блокировать и удалять пользователей OWNER и CLIENT

OWNER
    Добавляет автомибили
    Имеет возможность исключить из выдачи поиска автомобиль
    Может принять или отклонить заявку на аренду автомобиля
    Может прикреплять к автомобилю картинки и видео

CLIENT
    Указывает информацию о себе, включая данные о водительском удостоверении
    Получает список доступных автомобилей
        имеется поиск по нескольким полям сразу: производитель, модель и год
        имеются фильтры по отдельным характеристикам тип автомобиля, мощность, цена
        имеется возможность сортировки по различным характеристикам цена аренды, мощность, год выпуска....
    Выбирает подходящий вариант и создаёт заявку на аренду
    После успешного завершения аренды, может оставить отзыв в виде текста и оценки
